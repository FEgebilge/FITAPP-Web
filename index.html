About the Project

FITAPP is an AI-powered fitness application developed as a senior graduation project. Built with Flutter and Spring Boot, it integrates MediaPipe for 33-point pose detection and machine learning for exercise classification. Using a modular architecture, it delivers real-time feedback via WebSocket technology and stores detailed workout data securely for analytics. This project combines cutting-edge AI techniques with practical system design to provide accurate and scalable fitness tracking for real-world use.
